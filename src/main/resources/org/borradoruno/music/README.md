# Música de fondo

Coloca aquí los archivos `.mp3` con los siguientes nombres exactos:

- `menu.mp3`  — música del lobby, registro y sala (loop infinito)
- `game.mp3`  — música durante la partida (loop infinito)

Recomendaciones:
- Duración 1-3 minutos (se reproduce en loop)
- Bitrate: 128-192 kbps (suficiente para música ambiente)
- Volumen: bajar el master del MP3 ~6dB para que no opaque los efectos

Fuentes gratuitas:
- pixabay.com/music
- freemusicarchive.org
- youtube.com/audiolibrary (descargas legales)
